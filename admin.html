Setup Guide
Step 1 — Firebase Project

Go to console.firebase.google.com → Create project → Name it "radiance".
Click the </> web icon → Register app → Copy the config object.

Step 2 — Paste Firebase Config

Open js/firebase-config.js and replace the placeholder values with your actual Firebase config keys.

Step 3 — Enable Auth

Firebase Console → Build → Authentication → Get Started → Enable Google as a sign-in provider. Optionally also enable Email/Password.

Step 4 — Create Firestore Database

Firebase Console → Build → Firestore Database → Create database → Start in test mode (you'll tighten rules later).

Step 5 — Make Yourself Admin

Log in to the site with Google first. Then go to Firebase Console → Firestore → Add collection admins → document ID = your Firebase UID → add field admin: true.

To find your UID: Firebase Console → Authentication → Users → copy your UID.

Step 6 — Seed Products

Once Firebase is configured, click the "Seed Initial Data" button in the Products tab. This creates all 5 candles in Firestore so you can manage them from here.

Step 7 — Add Your Photos

Drop your candle images into the assets/ folder. The filenames expected are:
candle-hero.jpg · candle-about.jpg · candle-contact.jpg
You can use the same photo for all three for now.

Step 8 — Deploy to Vercel

Push the entire radiance/ folder to a GitHub repo. Go to vercel.com → Import repo → Deploy. Done. Your site is live.
